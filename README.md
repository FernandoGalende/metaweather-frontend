# FernandoGalende WeatherApp with React-Redux-Sagas-Sass-CssModules

This app is build with create-react-app.

## Run the project in local mode

And then you can run this app.
`npm install`

** Please do not forget add a CORS extension tool to your browser! **

`npm run start`

In the future, this test will work in others browsers, meanwhile please use  Google Chrome.

If you navigate to http://localhost:3000 you can enjoy this app.

## Tecnologies
React Redux Sagas Sass CssModules
